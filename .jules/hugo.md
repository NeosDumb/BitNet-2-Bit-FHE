## 2024-05-20 - O(1) Memory Mapping via Sign Invariance in Scale-based Quantization
**Learning:** In scale-based quantization (e.g., `quantize_i2_s`), computing the exact scale `max = max(|val|)` over an array and then mapping elements based on the sign `val * max > 0` has a redundant temporal dependency on `max`. If the scale `max` is non-negative, the evaluation mathematically simplifies to `val > 0`. Because of this sign invariance, we can actually compute the scale `max` at the same time as quantizing and bit-packing the elements into their target destination.
**Action:** Eliminate intermediate array heap allocations and multiple O(n) passes over memory by fusing scale calculation into the bit-packing loop when the quantization bucket boundary evaluates only the sign.
