## 2025-02-26 - O(1) Memory Mapping for I2_S Quantization
**Learning:** In scale-based quantization where the output value depends on `val * scale > 0`, if `scale` is strictly non-negative (like the maximum absolute value of a tensor), the condition simplifies mathematically to `val > 0`. This sign invariance allows determining the quantized bits independently of the global scale, removing the temporal dependency that normally requires allocating O(N) intermediate buffers.
**Action:** When evaluating sequential multi-pass algorithms over large tensors (like max-scan followed by quantization), look for invariant properties in the mathematical definitions. Eliminating intermediate state buffers follows the physical principle of least action, minimizing memory bandwidth usage and caching penalties.
