## 2024-05-15 - Independence of Sign Mapping from Scalar Magnitude
**Learning:** The quantization function mapping values to ternary states (-1, 0, 1) depends purely on the sign function $sgn(x)$ and zero-thresholding. Because the sign of a real number is invariant to scaling by a positive magnitude ($sgn(c \cdot x) = sgn(x)$ for $c > 0$), the bit-packing phase is strictly independent of the global maximum value. Thus, we discovered an anti-pattern: performing a separate pass to find the maximum, scaling values, and then mapping to ternary, whereas the ternary mapping can be performed simultaneously with the max-scan.
**Action:** Always verify if multi-pass scaling algorithms can be mathematically decoupled. By exploiting scaling invariance, multi-pass scans can be fused into single-pass operations, eliminating $O(N)$ intermediate heap allocations and halving memory bandwidth.
