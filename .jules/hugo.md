## 2025-02-18 - [O(1) memory mapping via sign invariance in quantize_i2_s]
**Learning:** In scale-based quantization (e.g., `quantize_i2_s`), calculating the absolute maximum determines a global scale that is strictly non-negative. This exposes a mathematical sign invariance property where `(val * scale) > 0` simplifies logically and identically to `val > 0`. This discovery eliminates the necessity to temporally sequence global scale determination ahead of ternary state evaluation.
**Action:** By applying this invariant property, three O(n) passes (max-scan, quantization, and bit-packing) can be functionally fused into a single O(n) pass, enabling an O(1) memory footprint during mapping by entirely bypassing intermediate array allocations. This insight should be systematically evaluated in any scalar-dependent quantizing pass to unearth similar single-pass vectorization and allocation bypass opportunities.
